ERRORES EN LA BUILD DEL APK Y SOLUCIONES
=========================================

ERROR 1: JAVA_HOME no configurado
----------------------------------
Causa: No se encontró Java en el PATH del sistema.
Solución: 
- Instalar JDK 17 (descargar de https://www.oracle.com/java/technologies/downloads/ o https://adoptium.net/)
- Configurar variable JAVA_HOME = C:\Users\Naru\jdk-17.0.2
- Agregar al PATH: C:\Users\Naru\jdk-17.0.2\bin

ERROR 2: Android SDK no encontrado
-----------------------------------
Causa: No se encontró el Android SDK en la ubicación esperada.
Solución:
- Opción A: Instalar Android Studio (https://developer.android.com/studio)
- Opción B: Descargar Android cmdline-tools:
  1. Descargar de: https://dl.google.com/android/repository/commandlinetools-win-11076708_latest.zip
  2. Extraer en: C:\Users\Naru\AppData\Local\Android\sdk\cmdline-tools\latest
  3. Ejecutar: sdkmanager --install "platforms;android-34" "build-tools;34.0.0"
  4. Crear archivo local.properties con: sdk.dir=C:\Users\Naru\AppData\Local\Android\sdk

ERROR 3: google-services.json no encontrado
-------------------------------------------
Causa: No existe el archivo de configuración de Firebase.
Solución:
- Si no usas Firebase: eliminar las líneas 47-54 del archivo android/app/build.gradle
- Si usas Firebase: descargar google-services.json desde Firebase Console

PASOS PARA COMPILAR EL APK
==========================
1. Asegurarse que JAVA_HOME esté configurado
2. Asegurarse que ANDROID_HOME esté configurado
3. Ejecutar: npx cap sync android
4. Ejecutar: cd android && gradlew assembleDebug
5. El APK se generará en: android/app/build/outputs/apk/debug/

NOTA: La descarga del cmdline-tools falló por timeout. Se recomienda descargar manualmente desde un navegador.

BUILD EN RENDER (CI/CD)
=======================
Para construir el APK en Render, necesitas configurar el environment:

1. En el dashboard de Render, ir a "Environment" del servicio
2. Agregar estas variables:
   - JAVA_HOME: /opt/java/openjdk (Render tiene Java por defecto)
   - ANDROID_HOME: /opt/android-sdk
   - ANDROID_SDK_ROOT: /opt/android-sdk

3. Crear script de build (build.sh):
```bash
#!/bin/bash
set -e

# Instalar Android SDK si no existe
if [ ! -d "/opt/android-sdk" ]; then
    mkdir -p /opt/android-sdk/cmdline-tools
    cd /opt/android-sdk/cmdline-tools
    wget -q https://dl.google.com/android/repository/commandlinetools-linux-11076708_latest.zip -O cmdline-tools.zip
    unzip -q cmdline-tools.zip
    mv cmdline-tools latest
    rm cmdline-tools.zip
    
    # Instalar plataformas y build-tools
    yes | /opt/android-sdk/cmdline-tools/latest/bin/sdkmanager --licenses > /dev/null 2>&1
    /opt/android-sdk/cmdline-tools/latest/bin/sdkmanager "platforms;android-34" "build-tools;34.0.0" "platform-tools"
fi

export ANDROID_HOME=/opt/android-sdk
export ANDROID_SDK_ROOT=/opt/android-sdk
export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin:$ANDROID_HOME/platform-tools

# Build del APK
cd /app
npm install
npx cap sync android
cd android
./gradlew assembleDebug
```

4. Agregar "Build Command" en Render: "./build.sh"
5. El APK estará en: android/app/build/outputs/apk/debug/
